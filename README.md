# Bleach: Final Substitute Soul Reaper

A fan-made **2D action game** inspired by the anime **Bleach**, built using **Python** and **Pygame**.

Fight your way through relentless Hollow encounters, unleash Ichigo's iconic abilities, and face **Sosuke Aizen** in the final battle. This project was developed as my **first complete game development project**, with a focus on learning gameplay programming, animation systems, AI, and software engineering principles.

# Features

-  Fast-paced 2D combat
-  Shikai & Bankai transformation system
-  Getsuga Tenshō projectile attacks
-  Visored Ultimate ability
-  Enemy AI
-  Menos Grande encounters
-  Final boss battle against Sosuke Aizen
-  Health & Ultimate Gauge
-  Story ending sequence
-  Background music and sound effects


# Technologies Used

# Installation

Download the latest release from the **Releases** section and run:

```
Bleach.exe
```

No Python installation or additional dependencies are required.
